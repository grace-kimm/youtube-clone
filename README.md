## URL
```
/ -> home
/join
/login
/search

/users/:id
/users/logout
/users/:id/edit -> edit my profile
/users/:id/delete -> delete my profile

/videos/:id -> Watch video
/videos/:id/edit
/videos/:id/delete
/videos/upload
```

## How to Use
1. `npm i`
2. `npm run dev`
3. connect to `http://localhost:4000/`
